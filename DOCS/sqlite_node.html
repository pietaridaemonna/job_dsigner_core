It will install all dependencies and make your project ready to work. # NPM INSTALL SQLite is transaction database engine, it means you can run DDL and DML queries of SQL. DDL queries deal with creation of database, tables, view etc where as DML queries are responsible for CRUD operation (insert,delete,update) etc. Here is a code to create SQLite database and performing SQL operations. ```bash vi /etc/mongod.conf ``` and change bindIp to your IP for external listening ```bash netstat -a |grep 27017 firewall-cmd --get-active-zone firewall-cmd --get-zones firewall-cmd --zone=external --change-interface=eth0 firewall-cmd --zone=home --list-services firewall-cmd --zone=external --add-port=27017/tcp service mongod restart netstat -a |grep 27017 ```
