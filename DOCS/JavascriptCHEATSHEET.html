# Javascript ## Enums ```javascript If you want the enum values to hold properties, you can add them to an extra object: var SizeEnum = { SMALL: 1, MEDIUM: 2, LARGE: 3, properties: { 1: {name: "small", value: 1, code: "S"}, 2: {name: "medium", value: 2, code: "M"}, 3: {name: "large", value: 3, code: "L"} } }; // Freeze the object. Object.freeze(SizeEnum); Then use it like so: var mySize = SizeEnum.MEDIUM; var myCode = SizeEnum.properties[mySize].code; // myCode == "M" ```
